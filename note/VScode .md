### VS code 快捷键



#### Ctrl

1. Ctrl + /： 注释
2. Ctrl + \：快速分屏
3. Ctrl + [：左缩进
4. Ctrl + ]：右缩进
5. Ctrl + D：匹配相同单词
6. Ctrl + L：选中当前行
7. Ctrl + Enter：在当前行下方插入一行
8. Ctrl + Shift + Enter：在当前行上方插入一行
9. Ctrl + Shift + L：光标批量操作



#### Alt

1. Alt + Up：把光标所在行向上移动一行
2. Alt + Down：把光标所在行向下移动一行
